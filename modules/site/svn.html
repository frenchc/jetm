JETM SVN Access

The project subversion repository is currently hosted at SourceForge.net. Before 1.2.0 it was using CVS at Sourceforge. During the transition the source structure was changed significantly, therefore SVN is missing the change history of CVS.

As usual anonymous SVN access is available. Information about accessing the SVN repository may be found in the document titled Subversion (Version Control for Source Code).

Anonymous SVN Access

The JETM subversion repository can be checked out through SVN with the following instruction set: 

JETM tree:  svn co https://jetm.svn.sourceforge.net/svnroot/jetm jetm
  
JETM trunk: svn co https://jetm.svn.sourceforge.net/svnroot/jetm/jetm/trunk jetm

SVN Web Access

You may also view the complete history of any file in the repository using the web-based Subversion interface.
