JETM release plan

This is the release plan for JETM. Intermediate releases are available through SVN and snapshots, release dates aren't fixed yet.

  • Improve Java EE integration
  • Further framework integrations (JBoss Seam, ...)
  • Improve graphing capabilities
  • Consolidate documentation

Planned features

  • Switch to event based monitoring (probably 2.0)
  • Google GWT Rendering
  • JBoss AOP Framework support (static, dynamic)
